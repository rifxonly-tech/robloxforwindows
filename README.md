# Roblox Redirect Launcher (`roblox.exe`)

## Overview
`roblox.exe` is a lightweight launcher whose purpose is to **redirect users to the official Roblox page on the Microsoft Store** so they can download and install Roblox directly from there.

> Note: This file is **not Roblox itself**. It only opens Microsoft Store to the correct Roblox listing.

## How to Use
1. Download `roblox.exe`.
2. Run `roblox.exe` (double-click).
3. The app will **automatically open Microsoft Store** on the Roblox page.
4. Click **Get / Install** in Microsoft Store to download and install Roblox.

## What Happens When You Run It?
When executed, `roblox.exe`:
- Launches a Microsoft Store deep link (Store redirect)
- Sends the user to the official Roblox listing on Microsoft Store
- Leaves the download and installation process to Microsoft Store

## System Requirements
- Windows 10 or Windows 11
- Microsoft Store available and able to open
- Active internet connection

## Troubleshooting
### Microsoft Store does not open
- Ensure Microsoft Store is installed and not disabled by device policies (e.g., school/work restrictions).
- Try opening Microsoft Store manually and searching for **Roblox**.
- If the Store is broken, try updating Windows or resetting Microsoft Store:
  - Settings → Apps → Microsoft Store → Advanced options → Reset

### Windows SmartScreen warning
Windows may warn users about unfamiliar `.exe` files.
- Only run `roblox.exe` if you downloaded it from a trusted source.
- Consider scanning the file with antivirus software before running it.
